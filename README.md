# cordova-plugin-alt-beacon
Plugin to get beacon data

## Platforms
only for Android

## Installation
```
cordova plugin add https://github.com/vienom/cordova-plugin-alt-beacon
```

## Usage
```javascript
window.startRangingBeacons(function(beacons){
  console.log(beacons);
});
```
